# ⚽ FIFA World Cup Match Analysis (1930–2014)

A comprehensive Exploratory Data Analysis (EDA) project on FIFA World Cup matches from **1930 to 2014** using Python. This project uncovers historical trends, team performances, goal statistics, attendance records, and tournament insights through data visualization and statistical analysis.

---

## 📌 Project Overview

The FIFA World Cup is the world's biggest football tournament, producing decades of exciting matches and unforgettable moments. This project analyzes historical World Cup match data to answer questions such as:

- Which teams have performed the best?
- How has goal scoring changed over time?
- Does the host nation have an advantage?
- Which World Cup had the highest attendance?
- How have penalty shootouts evolved?
- Can teams be grouped based on their playing style?

The notebook demonstrates the complete data analysis workflow—from data cleaning to visualization and basic machine learning.

---

## 📂 Dataset

The dataset contains FIFA World Cup match records from **1930–2014**, including information such as:

- Match Date
- Stadium

- Attendance

- Match Outcome

## 📈 Key Insights

- Average goals per match have gradually decreased over the years.
- Host countries generally perform better during tournaments.
- Brazil remains one of the most successful teams in World Cup history.
- Penalty shootouts have become increasingly common in modern tournaments.
- Attendance has little direct impact on the number of goals scored.
